SCENARIO:

Network is Unavailable

DESCRIPTION:

This scenario tests the effects of network loss on the Focus application and environment. This will be simulated by disconnecting the network cable from the Focus 186 Box. The network cable will be disconnected for 40 minutes, then reconnected for 40 minutes.

TIMELINE:

{timeSegment1}: Steady State

{timeSegment2}: Network is Unavailable

{timeSegment3}: Network is Available, healing

Total Duration: 1 hour, 40 minutes

OBSERVATIONS:

  • Due to the nature of this test, AppD metrics are not available for the Server during the portion of the test where network loss was simulated.
  • Orders were not able to be placed during the time Network connectivity was lost.
  • The environment returned to normal very quickly after reconnected to the Network.
  • Overall Results of Chaos test was successful, and achieved results were as expected.
